Generative Building Block Approach

Simple

Less is more. Easier to fully understand. Easier to reason about. Easier to replace and adapt. When done thoughtfully, simple is more generative.

Composable

Building blocks are composable when they have an API, and even more composable when implemented with common interfaces and protocols.

Extensible

Whether via hooks, plugins, or scripting, allowing building blocks to be modifiable keeps its core design focused while supporting exceptional cases.

Ergonomic

None of this matters unless the experience is human-friendly. APIs and interfaces that make sense and feel natural.
